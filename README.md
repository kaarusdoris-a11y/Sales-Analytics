## 🇬🇧 In English

### Project Description

This repository contains the analysis of **UrbanStyle.ltd** fictional sales data, created as part of the DACA (Data Analyst Career Accelerator) programme. Our goal is to clean, analyse, and present sales data in a clear and accessible way.
